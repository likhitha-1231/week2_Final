# Project3_Smart_Analytics_Tool

## Project Description
This project, **Smart Analytics Tool**, provides a comprehensive analysis of global COVID-19 trends. It processes daily report data to extract meaningful statistics, enabling users to visualize and monitor the impact of the pandemic through interactive charts and insights.

## Dataset Link
* **Dataset Name:** COVID-19 Daily Report (Jan 01, 2021)
* **Link:** (https://raw.githubusercontent.com/CSSEGISandData/COVID19/master/csse_covid_19_data/csse_covid_19_daily_reports/01-01-2021.csv)
